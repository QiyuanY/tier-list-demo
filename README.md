# 层级排行榜系统

一个现代化的层级排行榜应用，支持拖拽交互和自定义等级分类。

## 功能特性

- 🎯 **分级系统**: S、A、B、C、D五个等级
- 🎨 **个性化标签**: 夯、顶级、人上人、NPC、拉完了
- 🎮 **拖拽交互**: 直观的拖拽操作
- 💾 **本地存储**: 自动保存排行榜状态
- 📱 **响应式设计**: 支持多种屏幕尺寸

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **交互**: HTML5 Drag API
- **存储**: LocalStorage
- **样式**: 现代化UI设计

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/QiyuanY/tier-list-demo.git
cd tier-list-demo
```

2. 在浏览器中打开 `index.html`

3. 开始使用拖拽功能！

## SVG预览

查看 `tier-list-demo.svg` 文件获取完整的界面预览图。

## 等级体系

- **夯** (S级) - 最高等级，红色标签
- **顶级** (A级) - 高等级，橙色标签
- **人上人** (B级) - 中等级，黄色标签
- **NPC** (C级) - 低等级，灰色标签
- **拉完了** (D级) - 最低等级，白色标签
